Healthcare Center Full Pack
==========================

This package contains:
- Frontend (browser-only) with LocalStorage demo
- PHP backend stubs and SQL schema for optional migration
- README and basic instructions

To run frontend-only:
- Unzip and open index.html in your browser. Demo accounts are pre-seeded in localStorage.

To use PHP backend:
- Import php-backend/install.sql into MySQL
- Place php-backend in your webroot and update config.php with DB credentials
- Implement API logic and connect frontend to backend endpoints

Security: This is a demo. Do NOT use in production without proper security hardening.
